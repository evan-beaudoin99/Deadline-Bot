Deadline Bot 3000

This is a beta testing version for an application that sends weekly updates to users on when their Assignment, Tests/Quizes, Midterms and Tutorials are due. User's are required to create a user account on this site and then drap and drop the pdf's of their course outlines. The data is parsed using gpt 4.2 into json. The user will then be sent weekly updates on a complete summary of what is going on for that week for every single on of their classes, all in one place!
