# DEADLINE BOT

Tired of constantly haveing to reopen course outlines to remember when that one assignment is due? 

Look no more than Deadline Bot!

Deadline Bot sends you weekly updates and foresight on all your deadlines (Tests, Quizzes, Exams, Assignments, Tutorials, etc)

All you need to do is make an account, then drag and drop your course outlines once! Simple!
